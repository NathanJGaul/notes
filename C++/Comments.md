# Comments

## Single Line Comments

```
// This is a single line comment
std::cout << "Hello World!\n";	
```
```
std::cout << "Hello World!\n"; // This is also a single line comment
```

## Multi Line Comment

```
/*
multi
line
comment
*/
```
